# User Roles

## Super Admin
- Manage entire system
- Manage users
- View reports

## Operations Manager
- Create companies
- Generate HR accounts
- Monitor verification workflow

## Client HR
- Create employee forms
- Upload documents
- Track verification status

## Verification Executive
- Accept tasks
- Verify employees
- Track calls
- Update status

## Accountant
- Manage invoices
- Track payments
- Generate reports
