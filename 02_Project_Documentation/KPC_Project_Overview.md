# KPC Verification Management System

## Project Overview
KPC Verification Management System is an enterprise workflow management platform used to manage employee verification requests submitted by client companies and processed by KPC verification executives.

## User Roles
1. Super Admin
2. Operations Manager
3. Accountant
4. Verification Executive
5. Client (Company HR)

## Key Modules
- Company Management
- HR Account Management
- Employee Verification Forms
- Executive Task Assignment
- Verification Tracking
- Reports & Notifications
